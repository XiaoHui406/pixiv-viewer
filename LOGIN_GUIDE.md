# 登录指南

## 推荐登录方式

### Pixiv ID 和密码登录（推荐）

在本应用中，我们强烈建议您直接使用 Pixiv ID 和密码进行登录，这是最稳定和兼容的登录方式。

#### 登录步骤：
1. 打开应用后，进入 Pixiv 登录页面
2. 输入您的 Pixiv ID 和密码
3. 完成登录

### 为什么推荐这种方式？

1. **兼容性更好**：避免了第三方登录的安全策略限制
2. **稳定性更高**：在 WebView 环境中表现更稳定
3. **更安全**：避免了 OAuth 流程中的潜在风险

## 第三方登录问题说明

### Google 登录问题

如果您尝试使用 Google 登录，可能会遇到以下错误：

```
禁止访问:"pixiv"的请求不符合Google的相关政策
"pixiv"的请求不符合Google的"使用安全浏览器"政策。
错误403: disallowed_useragent
```

**原因**：
- 这是由于 WebView 环境的 User-Agent 不符合 Google 的安全策略
- Google 检测到当前环境不是标准浏览器，因此拒绝了 OAuth 请求

### 其他第三方登录

类似的问题也可能出现在 Twitter、Apple ID 等第三方登录方式中。

## 故障排除

### 如果无法登录

1. 确认网络连接正常
2. 检查账户信息是否正确
3. 尝试清除 WebView 缓存
4. 重启应用后重试

### 其他问题

如果您遇到其他登录问题，请尝试：
- 使用移动数据网络或更换 WiFi 网络
- 检查设备时间和时区设置
- 更新应用到最新版本
